# Meta Ads Agent Dashboard

Dashboard en tiempo real conectado a Google Sheets.

## Cómo subir a Vercel

1. Subí todos estos archivos a un repositorio de GitHub
2. En Vercel, importá el repositorio
3. Framework: Vite
4. Deploy

## Estructura
```
meta-ads-agent/
├── index.html
├── package.json
├── vite.config.js
├── vercel.json
└── src/
    ├── main.jsx
    └── App.jsx
```
